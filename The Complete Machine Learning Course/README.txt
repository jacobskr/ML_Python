The Complete Machine Learning Course with Python on Udemy.com taught by Anthony NG and Rob Percival.

All data used is either in the Data folder or imported through a package (seaborn, ect).

Topics covered:
1. Regression
2. Classification
3. SVM
4. Trees
5. Ensemble Methods
6. kNN
7. Dimensionality Reduction
8. Clustering

Jupyter notebooks for each topic are found in the Projects folder.
